# Sign-language-to-audio-converter-
Sign language to audio converter has been made using LSTM, which is very good in learning sequential datas. This project helps to reduce the communication barrier between hearing impared and normal people. The main aim of this project is to make a user friendly system that can be used by people who cant hear with.Here python has been used as the main programming language and we have also OpenCV, Mediapipe, Tensorflw kera, Scikit-learn,NumPy,Pandas Text-To-Speech converter. The model first captures the video input using webcam, then this data is processed using mediapipe, then the hand landmarks are detected, then the coordinates are detected. After that the signs are converted into text and at last using text to speech converter the text is converted into speech.
